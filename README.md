# Insurance Policy Data Simulation & Claims Analysis

This project simulates insurance policy sales and claims data using Python and Pandas.

## Features
- Generated 1,000,000 policy records
- Simulated claims based on business rules
- Calculated premium, claim cost, and loss ratio
- Performed analysis by tenure and month

## Tools Used
- Python
- Pandas
- NumPy

## Files
- generate_data.py → data simulation
- analysis.py → analytical queries
- claims_data.csv → generated claims data
- explanation.txt → approach and assumptions

## Purpose
This project was built as a Business Intelligence / Data Analytics assignment to demonstrate data simulation, data processing, and analytical skills.
